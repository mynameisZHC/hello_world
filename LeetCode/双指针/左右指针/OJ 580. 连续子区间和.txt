题目描述
给一串含有c个正整数的数组, 求出有多少个下标的连续区间, 它们的和大于等于x。

解答要求
时间限制：1000ms, 内存限制：100MB
输入
第一行两个整数c x（0 < c <= 1000000, 0 <= x <= 100000000)
第二行有c个正整数（每个正整数小于等于100)。

输出
输出一个整数，表示所求的个数。

样例
输入样例 1 复制

3 6
2 4 7
输出样例 1

4
提示样例 1

note:
1:双指针

code:
long long ContinueSubArrSum(long long targetNum, const vector<int> &array)
{
    long long result = 0;
    int leftIndex = 0;
    int rightIndex = 0;
    long long tempSum = 0;
    // 这里只能是leftIndex < array.size() 不能加rightIndex < array.size()
    while (leftIndex < array.size()) {
        // 一直移动右指针 直到满足条件或者到达边界
        while (rightIndex < array.size() && tempSum < targetNum) {
            tempSum += array[rightIndex];
            rightIndex++;
        }
        // 在tempSum >= targetNum情况下才进行result更新
        if (tempSum >= targetNum) {
            result += array.size() - rightIndex + 1;
        }
        // 更新tempsum 左指针向右移动
        tempSum -= array[leftIndex];
        leftIndex++;

    }
    return result;
}
